The IsThisFirstTime_Log_From_LiveServer flag controls access to the entire ML directory:

  • When TRUE: All ML pages are blocked and users are redirected
  • When FALSE/NULL: Normal access to ML directory is allowed
  • Scope: Affects all pages within the ml/ directory
